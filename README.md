# Instagramカルーセル投稿（4:5縦長）一括生成ツール

GitHub Pagesで動作する、**Instagramカルーセル（1080×1350 / 4:5）10枚**を一括生成・書き出しできるツールです。

## できること
- **ジャンル別テーマ切り替え**（教育 / プログラミング / 自己啓発）
- **スライド1**：タイトル・サブタイトル
- **スライド2〜9**：テキスト入力（改行OK）
- **一括貼り付けモード**：改行ごとにスライド2→9へ自動分配
- **キャプション自動生成**＋コピー
- **書き出し**：html2canvasでPNGを生成
  - **ZIPで一括保存**（推奨）
  - **連続ダウンロード**

## 使い方
1. `index.html` をブラウザで開く
2. 左パネルでジャンル選択・各スライド文言を入力
3. 必要なら「一括貼り付けモード」にGemini出力を貼り付けて「分配する」
4. 「全10枚を一括ダウンロード」

## GitHub Pagesで公開
1. このフォルダをGitHubにpush
2. GitHubの `Settings` → `Pages`
3. `Deploy from a branch` を選択し、`main` / `root` を指定して保存

## 補足
- 書き出しは **scale: 2** で実行し、文字のボケを抑えています。
- 画像は各スライドごとに `PNG` で生成されます。
